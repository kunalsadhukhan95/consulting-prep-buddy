# Your Consulting Preparation Buddy

A complete 7-tab consulting interview preparation website hosted on GitHub Pages.

## Files to upload to GitHub
- index.html, industry.html, frameworks.html, cases.html
- guesstimates.html, tips.html, guidance.html
- styles.css, chatbot.js
- cases_data.js, guesses_data.js, industry_data.js

## Deploy in 4 steps
1. Create a free GitHub account at github.com
2. Create a new PUBLIC repository named: consulting-prep-buddy
3. Go to Settings → Pages → Source: Deploy from branch → main → Save
4. Upload all files above → Commit changes

Your site goes live at: https://YOUR-USERNAME.github.io/consulting-prep-buddy/

## Chatbot
The chatbot widget appears on every page. When visitors type a question,
it responds with guidance and a direct "Book on Topmate" button,
driving them to topmate.io/kunal_sadhukhan for 1-on-1 sessions.
No API key or server setup needed — works out of the box.
